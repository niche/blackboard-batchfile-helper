blackboard-batchfile-helper by niche

creates blackboard enrolment batchfiles from lists of users and modules.

As a Blackboard administrator, I have spent hours creating batch files to enrol users to courses. I eventually gave in and created this script to do it for me. When you load this up, it will show two text areas and some options below. A list of users goes in the left hand column (one to each line) and a list of modules goes in the right hand one. Set the options below, then press “Generate Batch File”. It will then generate you a batch file for putting every user in the list onto every module in the list. Hint: copying and pasting columns from Excel into these text areas works really well There is some further processing that goes on to convert student numbers from our* student records system format (ASIS) to Blackboard user format (yes, they’re different!). It also has an option to put a year code onto the end of modules (e.g. -0910). If this doesn’t suit your requirements or you want to make it more configurable – be my guest!

Dependencies

PHP

Install

Drop these files onto your server, create a server writable folder called files in the folder you're installing this in. Load up index.php and have a play. You may then want to alter bb-batch.php to suit your preferences.
